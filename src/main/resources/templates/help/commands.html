ServoBot Commands

Command Properties

  • Permission: Who is allowed to trigger the command. The levels are
  • Hide: Whether 🔑 or not 🔒 the command is shown on the website. Use this to hide commands that most viewers aren't able to access, have sensitive content, or a way to provide 'Easter Eggs' for viewers to find.
  • Live: Whether the command is only available while streaming 📺 or can be triggered at any time 🕰️.
  • Twitch: Whether the command can be triggered from Twitch chat.
  • Discord: Whether the command can be triggered from the Discord server.

Command Types

Type Parameters Description
Respond Command Replies with the command's predefined text. The text can contain expressions.
Message Channel Command Replies with the command's predefined text to the predefined channel. The text can contain expressions.
Add Command !<command‑name> [<flags>] <text> Creates a new Respond Command with the given text. The command is triggered by command-name. The flags can change the permissions and settings of the command.

The add command can also change an existing trigger to respond with the new command, if no other triggers reference the previous command then it is deleted.
Delete Command !<command‑name> Deletes the trigger command-name. If there are no other triggers referencing the command then it also deletes the command.
Add Statement Command <book‑name> <text> Creates a new statement with the given text and adds it to the book named book-name.

If there is no book named book-name and the command was issued by a mod, then it also creates a new book with that name.
