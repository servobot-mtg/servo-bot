Quick Start Guide

To have the bot join your channel, you must log in.

You have a special invite to have the bot join your channel.

Logging Into the Website

The ServoBot website uses Twitch to authentication. Click the Twitch icon in the upper right of the page.

Using an Invite

Once you have an invite fill out the Bot Invitation form It asks you to set your time zone and to configure some commands. You can decide which commands you want enabled for your channel and change the command names.

Adding New Commands

There are two ways to add new commands.

  1. In chat on Twitch. Type the line: !add !<command-name> <command-text>
    • Note: If you changed the add command's name when submitting the invite then use that instead of !add.
    • Command-name must be composed of letters, numbers, and/or underscores.
    • Command-text is what the bot will respond with when someone in chat issues the command.
  2. On the ServoBot web page.
    1. Go to the channel's homepage.
    2. Sign in by clicking the Twitch icon in the upper right.
    3. Press the add button at the bottom of the table of commands. It will show a small dialog.
    4. Fill out the dialog.
      • Options are explained on the Command Help page.
      • For a Respond Command the Text option is the same as command-text above.
    5. Click the "Create Command" button. This will add your new command to the command table.
    6. On the command's row click the add button in the trigger column.
    7. Enter the command-name to trigger the command and then click the add button to submit.
